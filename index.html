About this Demo

This is a simple demonstration of continuous deployment from GitHub to a local Docker container using n8n as the automation tool.

When changes are pushed to the GitHub repository, an n8n workflow is triggered that:

  1. Pulls the latest code from GitHub
  2. Builds a new Docker image
  3. Runs the updated container on the local system
